# 🎧 HearMe

HearMe ek smart platform hai jo users ko lost items track karne aur recover karne me help karta hai.

## 🚀 Features
- 🔍 Lost item reporting
- 🔐 Secure identification system
- 🤖 Smart assistant support
- 📱 Simple and user-friendly UI

## 🛠️ Tech Stack

- CSS
- JavaScript
- (Optional: Add backend if used)

## 📌 How to Use
1. Website open karo
2. Lost item details fill karo
3. Hidden questions set karo (security ke liye)
4. Claim process track karo

## 💡 Purpose
Users ko unka lost wallet ya item easily recover karne me help karna — without stress.
